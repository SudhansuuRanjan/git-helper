What is Git?

Git is a free, open source distributed version control system tool designed to handel everything from small to very large projects with speed and efficoiency.It was created by Linus Tarvalds in 2005 to develop Linux Kernel. Git has the functionality, performance, security and flexibility that most teams and individual developers need. It also served as an important distributed version -control DevOps tool.

What is GitHub?

GitHub is code hosting platform for version control and collaboration based on git. It lets you and other work together on projects from anywhwere.
